## Análise de Vendas com SQL

Este projeto foi desenvolvido com o objetivo de praticar consultas em SQL utilizando um dataset real de vendas.

A ideia principal é explorar os dados e extrair informações relevantes sobre desempenho de produtos, comportamento de clientes e distribuição de vendas.

## Sobre os dados

Os dados utilizados neste projeto são públicos e representam registros de vendas contendo informações como data do pedido, cliente, região, categoria, produto, quantidade vendida, receita e lucro.

Fonte:
https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting

## Análises realizadas

As consultas foram desenvolvidas com foco em responder perguntas básicas de negócio, como:

- Qual o total de vendas e lucro
- Quais produtos geram mais receita
- Como as vendas se distribuem por região
- Quem são os clientes com maior volume de compras
- Evolução das vendas ao longo do tempo
